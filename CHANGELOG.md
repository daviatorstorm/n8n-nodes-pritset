# Changelog

All notable changes to this project are documented in this file.

## 0.1.3 - 2026-07-17

### Fixed

- Replaced scaffold placeholder documentation links with the public Pritset node repository URLs.

## 0.1.2 - 2026-07-17

### Fixed

- Renamed the Trusted Publishing workflow to the Creator Portal-required `publish.yml` filename.

## 0.1.1 - 2026-07-17

### Fixed

- Published a new patch version so the n8n Creator Portal can rerun its automated checks.
- Confirmed public provenance metadata and repeatable node and credential builds.

## 0.1.0 - 2026-07-17

### Added

- Pritset API credentials using `Authorization` and `X-Secret` headers.
- PDF generation in direct mode with binary output.
- Asynchronous PDF generation with webhook delivery.
- Template ID and JSON template data support for incoming workflow items.
- n8n node-picker metadata and Pritset API documentation links.
- Public npm access metadata for provenance-enabled first publication.
- Repeatable clean builds that always emit node and credential files.
